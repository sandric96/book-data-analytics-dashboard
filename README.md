# 📚 Book Data Analytics Dashboard

End-to-end data analysis project using Python and Power BI.

---

## 🎯 Objective

To analyze a books dataset and extract insights about:
- Ratings distribution
- Publishing trends
- Top authors and publishers
- Business patterns in publishing

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy)
- Power BI
- Jupyter Notebook

---

## 📊 Workflow

### 1. Data Cleaning (Python)
- Removed duplicates
- Parsed and standardized dates
- Created publication year feature

### 2. Exploratory Data Analysis
- Rating distribution
- Correlation analysis
- Trend analysis

### 3. Power BI Dashboard

The dashboard consists of 3 pages:

#### 🟣 Executive Overview
- KPIs
- Top authors
- Books over time

#### 🟦 Ratings Analysis
- Rating distribution
- Pages vs rating
- Ratings vs popularity

#### 🟩 Publishing Insights
- Top publishers
- Average rating by publisher

---

## 📈 Key Insights

- Most books have high ratings
- A small number of publishers dominate production
- No strong correlation between pages and rating
